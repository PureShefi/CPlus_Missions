========================================================================
    CONSOLE APPLICATION : Bar Project Overview
========================================================================

The project demos a bar named puzzles.
In it there are two types of drinks, wine and beer.
you can list options with 0,
		see how i prepared your last drink with 99
		exit bar with 100.

There are three classes which hold the drinks:
Drink - abstract class of a drink
Beer - class of a beer
Wine - class of a wine
